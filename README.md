# ThinkPHP5-ClassRoom

## 功能预览
### 后台
#### 教学楼和教室管理
#### 课程表管理
  （利用php 的curl实现爬虫功能，自动从教务管理批量获取）
#### 用户管理
#### 教室申请管理
#### 系统管理
### 前台（使用组件 bootstrap ，H-UI admin，jquery 等）
#### 用户个人信息
#### 申请使用教室
#### 学生教务网账号绑定（利用爬虫实现类似超级课程表功能，查询个人课表，成绩）
#### 用户申请管理
#### 用户申请状态查看（phpmailer）
